How this works and what this does

Once the "Enable GPS Tracking" button is clicked, the code starts to run. The code constantly detects your current location and displays your most recent location on a map (hence the tracking). However, you must understand that the code does NOT send anything to the server or any third-party due to the nature of Javascript. To send information to a server or third-party, another coding language is required (and I know little other than HTML, CSS and Javascript, neither of which stores and sends data elsewhere).

If something occurs that prevents the code from tracking your location, it will throw an error and display it to the user (e.g. Try turning off the wifi and enable GPS Tracking).

There is also one very special feature: theoretically, if you reach Ocean Park, this website/app will alert to you once stating that you have arrived at Ocean Park. However, this is yet to be tested since I do not have enough time to bring my whole computer to Ocean Park to check it out.
